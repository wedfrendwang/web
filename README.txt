在js的实践中，相应的语言结构也已经相应掌握，只是从不会到会容易，但是从会到精这段路又是何其的漫长。

之前有过一段话，叫做，当我们无法扩大生命的长度那就请增加它的宽度，思想的厚度


但是工作上希望自己能够脚踏实地，虚心向前，一步一步。我们都向往高薪，轻松的生活。为自己的理想而工作。


之前看到一个问题：怎么样才能将为了生活编程进化成为了理想编程？？？

池建强老师说：当钱足够的时候便可以为了理想编程。前提是你的能力能否为你赚到足够的钱？？？说的庸俗一点，每一个基层的工作者都希望将自己的能力转化为钢镚，但是当能力不足的时候，还是先默默积攒能量，厚积薄发。当机会真的降临，我们可以不让它逃脱。


自己是一个晚熟的人，年轻少不更事喜欢玩，学业看的很轻，以至于现在有点吃力。但是也感谢自己的经历，这样的生活才算完整，至少自己过得不算无趣，生活的不算太烂，人品上不算太差，还有着自己的梦。
